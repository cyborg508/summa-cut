summa-cut

Przygotowanie PDF do druku i cięcia bez ręcznej gimnastyki

Desktopowy program do układania użytków, rozdzielania druku i wykrojnika, podglądu arkusza i eksportu gotowych plików druk.pdf oraz wykrojnik.pdf.

Zobacz status projektu Funkcje

Po co powstał summa-cut

Program powstaje z praktycznej potrzeby produkcyjnej: szybciej przygotować PDF-y do wydruku i cięcia na ploterze, bez powtarzalnej ręcznej roboty i bez mieszania ustawień między drukiem a wykrojnikiem.

To nie jest ogólny kombajn DTP. Kierunek jest prosty: narzędzie do konkretnego, technicznego workflow poligraficznego.

Aktualny fokus

  • PDF wejściowy dla druku i wykrojnika
  • układ użytków na arkuszu
  • OPOS-y i bezpieczne pole robocze
  • eksport gotowych plików produkcyjnych

Co już działa

Niezależny wybór druku i wykrojnika

Można wskazać osobny plik i osobną stronę dla druku oraz wykrojnika, także przy wielostronicowych PDF-ach.

Podgląd i układ arkusza

Program liczy prosty layout siatkowy, pokazuje podgląd druku, wykrojnika i całego arkusza oraz umie porównać wariant 0° / 90°.

Eksport produkcyjny

Gotowy wynik zapisuje jako dwa pliki: druk.pdf oraz wykrojnik.pdf.

Tryb z odstępami

Układ działa jako kafel użytek + odstęp, z centrowaniem i przycinaniem obu źródeł do wspólnego modułu.

Tryb bez odstępów

Program układa druk na styk i generuje siatkę linii wykrojnika wychodzących 1 mm poza skrajny obrys.

Tryb specjalny

Osobny prototyp do nieregularnych kształtów: trimowanie druku do rzeczywistego obrysu wykrojnika + spad.

Status projektu

Projekt jest na etapie roboczego MVP. Najważniejsza baza już działa, ale to jeszcze nie jest finalna wersja handlowa.

  • działające GUI desktopowe w Pythonie / PySide6
  • drag & drop PDF
  • bounding box z pliku druku
  • OPOS-y 2 × 2 mm
  • eksport dwóch PDF-ów wyjściowych
  • prototyp trybu specjalnego z clipping path

Najbliższe kroki

  1. testy na realnych plikach produkcyjnych
  2. dopracowanie montażu 2×2 i sterowania wzorcem
  3. lepsza ergonomia edytora trybu specjalnego
  4. przygotowanie paczki Windows do łatwego wdrożenia

Pobieranie
