# summa-cut — lokalny szkic GitHub Pages

Przygotowane pliki strony:
- `docs/index.html`
- `docs/style.css`

## Co to jest
To prosty, statyczny szkic strony projektu pod GitHub Pages.

## Jak opublikować później
1. wrzuć projekt do repozytorium GitHub, np. `summa-cut`
2. wejdź w **Settings → Pages**
3. w sekcji **Build and deployment** ustaw:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` albo `master`
   - **Folder:** `/docs`
4. zapisz ustawienia
5. po chwili strona pojawi się pod adresem GitHub Pages repozytorium

## Co warto dodać przed publikacją
- 2–4 screenshoty programu
- link do pierwszego publicznego wydania Windows
- krótki changelog
- sekcję kontaktową albo formularz zgłoszeń

## Uwaga
Na razie sekcja pobierania jest celowo placeholderem, bo nie ma jeszcze publicznej paczki release.
